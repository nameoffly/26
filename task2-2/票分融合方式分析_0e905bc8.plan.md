# 票分融合方式对比分析方案

## 项目背景

使用 [combined_contestant_info.csv](task2-2/combined_contestant_info.csv) 中的预测粉丝投票百分比数据，对比分析"排名法"与"百分比法"两种票分融合方式。

## 数据结构

**数据字段**：赛季、周数、人名、伴侣名、地区、国家、年龄、行业、最终名次、比赛结果、评委评分百分比、粉丝投票百分比、总百分比

**数据范围**：34季全部数据（约2779行）

---

## 一、任务一：两种方法对比分析

### 1.1 核心分析逻辑

**分析范围**：所有34季（对每季同时应用两种方法进行模拟对比）

**百分比法**（S3-27实际使用）：

- 综合得分 = 评委百分比 + 观众百分比
- 淘汰综合得分最低者

**排名法**（S1-2, S28-34使用）：

- 综合排名 = 评委排名 + 观众排名（1=最好）
- 淘汰综合排名最大者

### 1.2 实现方案

在 `task2-2/` 目录下创建分析脚本 `voting_method_analysis.py`：

1. **按季度对比淘汰结果**：模拟两种方法在每周的淘汰决策，统计差异
2. **决赛排名准确性**：对比两种方法预测的决赛排名与实际排名
3. **偏向性分析**：当结果不同时，分析哪种方法更偏向观众投票

### 1.3 关键输出

- **整体一致性统计**：各季两种方法结果一致性（淘汰+决赛合并）
- **方法偏向统计**：排名法偏观众 vs 百分比法偏观众的次数对比
- **决赛排名准确性对比表**：所有34季的决赛排名准确性（完全正确、冠军正确、前两名正确）

## 二、任务二：争议选手分析

### 2.1 争议选手列表


| 季数  | 选手              | 特征       | 最终名次 |
| --- | --------------- | -------- | ---- |
| S2  | Jerry Rice      | 5周评委垫底   | 亚军   |
| S4  | Billy Ray Cyrus | 8周评委倒数第1 | 第5名  |
| S11 | Bristol Palin   | 12次评委最低  | 季军   |
| S27 | Bobby Bones     | 8周评委倒数第1 | 冠军   |


### 2.2 分析内容

1. **逐周追踪分析**：
  - 每周评委排名 vs 观众排名
  - 两种方法下的淘汰风险
  - 实际是否被淘汰
2. **方法对比**：
  - 同一争议选手在两种方法下是否会被淘汰
  - 淘汰时机是否不同

### 2.3 四种规则组合分析（重点）

**规则组合定义**：


| 规则编号 | 规则名称     | 融合方法 | 评委决定规则    |
| ---- | -------- | ---- | --------- |
| R1   | 纯百分比法    | 百分比法 | 无（直接淘汰末位） |
| R2   | 纯排名法     | 排名法  | 无（直接淘汰末位） |
| R3   | 百分比+评委决定 | 百分比法 | 末两位由评委选择  |
| R4   | 排名+评委决定  | 排名法  | 末两位由评委选择  |


**评委决定规则详解**：

- 先用综合得分/排名确定末两位
- 评委投票决定淘汰谁（模拟假设：评委淘汰评委分更低者）
- 此规则将应用到**所有赛季**进行模拟，而非仅S28之后

**分析方法**：

对每位争议选手（Jerry Rice S2、Billy Ray Cyrus S4、Bristol Palin S11、Bobby Bones S27）：

1. **逐周模拟**：在四种规则下分别计算该选手是否会被淘汰
2. **生存周数对比**：统计四种规则下选手能存活的周数
3. **决赛排名对比**：如果能进入决赛，四种规则下的预测排名
4. **关键转折点分析**：识别不同规则导致不同结果的关键周次

**分析维度**：

```
争议选手 × 4种规则 × 每周状态
├── 是否进入末两位
├── 是否被淘汰
├── 淘汰周数（若被淘汰）
└── 最终名次（若能完赛）
```

### 2.4 关键输出

- **四规则对比表**：每位争议选手在四种规则下的生存周数和最终名次
- **逐周状态追踪表**：详细记录每周每种规则下的淘汰风险
- **规则影响矩阵**：哪种规则对争议选手最有利/最不利
- **假设场景分析**：如果历史上采用不同规则，比赛结果会如何变化

## 三、任务三：方法推荐

### 3.1 评估维度（考虑节目组需求）

**核心原则**：节目组在一定程度上更乐见评委与观众的意见、投票结果出现分歧，因为这类争议能大幅提升观众的关注与参与热情。


| 评估维度      | 说明                      | 衡量指标                   |
| --------- | ----------------------- | ---------------------- |
| **争议适度性** | 是否能产生适度的争议（非极端），维持节目话题性 | 评委-观众分歧选手能否存活足够周数以制造话题 |
| **观众参与感** | 观众投票是否对结果有实质影响，维持投票热情   | 观众投票排名高的选手是否更容易晋级      |
| **戏剧效果**  | 是否能产生"爆冷"、"逆袭"等戏剧性结果    | 争议选手的最终名次是否出人意料        |
| **专业性体现** | 评委的专业判断是否得到一定程度的尊重      | 评委分数高的选手是否有合理的晋级优势     |
| **结果可控性** | 是否能避免完全"民粹化"的极端结果       | 评委决定规则对极端情况的干预效果       |


### 3.2 评估框架

**从节目组视角的核心问题**：

1. **话题制造能力**：哪种规则更容易产生"Jerry Rice式"或"Bobby Bones式"的争议明星？
2. **悬念保持能力**：哪种规则能让争议选手存活更久，保持观众持续关注？
3. **结果底线控制**：评委决定规则是否是一个合理的"安全阀"，在必要时介入？

### 3.3 推荐输出

基于分析结果，从以下角度给出推荐：

- **票分融合方式推荐**：哪种方法（百分比/排名）更符合节目需求
- **评委决定规则建议**：是否建议增设？在什么情况下启用？
- **争议平衡建议**：如何在专业性与娱乐性之间取得平衡
- **风险提示**：各种规则可能带来的极端情况及应对
